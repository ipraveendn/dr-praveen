# Dr. Praveen's Clinic - Token Booking and Live Queue

This project is a web-based platform for Dr. Praveen's endocrinology clinic, designed to manage patient token booking and provide real-time queue tracking.

## Features

*   **Patient Token Booking:** Patients can book a token for a clinic visit.
*   **Live Token Tracking:** Patients can track their position in the queue in real-time.
*   **Queue Management:** Staff can manage the patient queue from a simple dashboard.
*   **Real-time Updates:** The system provides real-time updates for the queue tracking and dashboard.

## Technologies Used

*   **Frontend:** React, Vite, React Router, Tailwind CSS
*   **Backend:** Node.js, Express
*   **Database:** Firebase Firestore (soon to be migrated to PostgreSQL)
